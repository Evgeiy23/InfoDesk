# InfoDesk — интеллектуальная RAG-система для сотрудников компании

**InfoDesk** — настольное приложение, созданное на **Python (PyQt)** с интегрированной **RAG-системой**, которое помогает сотрудникам быстро находить нужную информацию и получать понятные ответы на вопросы.  
Если система не может точно ответить, запрос автоматически **перенаправляется оператору**.

---

##  Основные возможности

- Поиск информации из корпоративной базы через **Embeddings** и **векторную базу данных**  
- Генерация ответов с помощью **LLM**  
- Автоматическое перенаправление сложных запросов оператору  
- Разделение ролей: **администратор**, **оператор**, **сотрудник**  
- Настройки профиля: смена имени, пароля, темы интерфейса  
- Хранение всех данных в **SQLite**

---

##  Архитектура RAG

```
Пользователь → Embeddings → Векторная БД → Контекст для промпта → LLM → Ответ
                                       ↓
                               "Перевожу на оператора"
                                       ↓
                                    Оператор
```

---

##  Технологии

- **Python 3.10+**
- **PyQt** — интерфейс  
- **SQLite** — база данных  
- **Embeddings + Векторная БД** — поиск релевантной информации  
- **LLM** — генерация ответов на основе контекста  

---

##  Как работает InfoDesk

1. Пользователь вводит вопрос в чат.  
2. Модуль **Embeddings** превращает вопрос в вектор.  
3. **Векторная БД** ищет похожие данные.  
4. Из найденной информации формируется **контекст для LLM**.  
5. **LLM** создаёт понятный ответ.  
6. Если ответа нет — система пишет *«Перевожу на оператора»*, оператор получает запрос и отвечает пользователю.

---

##  Роли пользователей

- **Администратор** — управляет аккаунтами сотрудников и операторов.  
- **Оператор** — получает и отвечает на сложные запросы.  
- **Сотрудник** — задаёт вопросы и получает ответы.

python -m venv venv
